Dashboard

Bienvenido al panel de control. Aquí puedes gestionar las citas, clientes, reservas y más, dependiendo de tu rol.

Roles y Accesos

Administrador

El Administrador tiene acceso completo a todas las funcionalidades del sistema. Puede gestionar las citas, reservas, usuarios, clientes y más. Además, tiene la capacidad de ver estadísticas y realizar configuraciones avanzadas del sistema.

  • Vistas: Panel de Control, Gestión de Citas, Gestión de Reservas, Gestión de Usuarios, Gestión de Clientes, Gestión de Reconocimientos Médicos.

Médico

El Médico tiene acceso a las citas asignadas y a los reconocimientos médicos realizados a los clientes. Puede visualizar y gestionar sus citas programadas, así como realizar los reconocimientos médicos correspondientes.

  • Vistas: Citas Asignadas, Detalles de las Citas, Historial de Reconocimientos Médicos.

Reserva

El rol de Reserva se encarga de gestionar las solicitudes de citas, coordinar horarios y médicos para cada cliente. Pueden ver todas las reservas solicitadas y asignarles un horario y médico disponible.

  • Vistas: Solicitudes de Reserva, Listado de Reservas.

Cliente

Los Clientes no tienen acceso a ninguna vista administrativa ni médica. Solo pueden interactuar a través de los encargados de reservas o médicos para gestionar sus citas.
